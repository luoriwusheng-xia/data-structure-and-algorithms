# 数组

### 寻找数组中心下标

> 给定一个整数数组 nums, 请编写一个能够返回数组 “中心下标” 的方法

中心下标是数组的一个下标， 其左侧所有元素相加的和 = 右侧所有元素相加的和

如果一个数组不存在中心下标， 返回 -1。 如果数组有多个中心下标， 应该返回靠近左边的那一个

注意： 中心下标可能出现在数组的两端

#### 解题思路

数组中某一个下标， 左右两边的元素之和相等， 改元素下标即为 数组的中心索引
思路： 先统计出整个数组的总和 `sum` , 然后从数组的第一个元素开始累加求和 `n1 + n2+ ... = total`； 总和 sum - 当前遍历的元素`current`，如果等于 total = sum - current ， 则当前的索引 i， 就是中心点下标

```javascript
输入  [1,3,4,7,3, 5]

左侧Sum： 1 + 3 + 4 = 8 . 右侧Sum： 3 + 5 = 8， 则中心点下标 i = 3
```
