# 寻找两个正序数组的中位数

[第4题](https://leetcode.cn/problems/median-of-two-sorted-arrays/)

给定两个大小分别为 m 和 n 的正序（从小到大）数组 nums1 和 nums2。请你找出并返回这两个正序数组的 中位数 。

算法的时间复杂度应该为 `O(log (m+n))` 。

### 提示

```javascript
nums1.length == m

nums2.length == n

0 <= m <= 1000

0 <= n <= 1000

1 <= m + n <= 2000

-10^6  <= nums1[i], nums2[i] <= 10^6
```
### 示例

示例 1：
```
输入：nums1 = [1,3], nums2 = [2]
输出：2.00000
解释：合并数组 = [1,2,3] ，中位数 2
```

示例2

```
输入：nums1 = [1,2], nums2 = [3,4]
输出：2.50000
解释：合并数组 = [1,2,3,4] ，中位数 (2 + 3) / 2 = 2.5
```


### 解析

* 时间复杂度为 `O(log (m+n))` 则表示需要使用 `二分查找`
* m + n >= 1  表示2个数组不会同时为空， 则中位数 一定存在
* 2个有序数组长度之和为`奇数`的时候， 中位数只有一个， 返回中间的那个数字 ； 长度位`偶数`时， 中位数有2个， 返回合并、排序以后位于`中间`的2个数的 `平均数` (x1 + x2) / 2

* 没有使用数组有序这一条件 （相当于全部重新排列了）， 时间复杂度取决于排序算法的时间复杂度

### 方式3：二分查找
中位数： 在只有一个有序数组的时候，中位数吧数组分割成`两个部分`

根据定义， 分情况，数组长度位奇数和偶数讨论

数组长度为偶数时， 中位数有2个，其中一个是左边数组的最大值，另一个是右边数组的最小值

中位数 = 2个有序数组的 第 (m + n) / 2  位置的数字

奇数： (m + n) / 2



```
例如： a1 = [1, 3] , a2 = [4]
则 m = 2, n = 1 ,总共数组： [1,3,4]

中间位置 = (m+n) / 2 = (2 + 1) / 2 = 1.5 , 向上取整， 为2， 就是说是数组的第2个元素，
则索引下标为1的值， list[1] = 3
```

偶数情况： (m + n) /2 , 跟 (m + n ) / 2 + 1
```
例如： a1 = [1, 3] , a2 = [4, 5]
则 m = 2, n = 12 ,总共数组： [1,3,4,5]

中间位置 = (m+n) / 2 = (2 + 2) / 2 = 2   第2个元素， 索引下标为 1
则索引下标为2的值， list[1] = 3， list[1 + 1] = 4

中位数 = (3 +4) / 2 = 3.5
```
