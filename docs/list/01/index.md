# 链表

## 链表反转

链表的特性是： 每个节点`Node` 只知道下一个节点是谁，不知道上一个是谁

- 链表有一个 next, 指向下一个`Node`
- 链表最后一个节点的 next 指向 `null`

```javascript
1->2->3->4->5

希望得到
5->4->3->2->1
```
