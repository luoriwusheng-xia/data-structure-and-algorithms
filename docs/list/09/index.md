# 素数个数统计

> 统计N以内的素数的个数

::: tip
素数： 只能被1和自身[整除]的自然数， 0、1 除外

如果一个大于1的正整数，除了1和自身外没有其他的约数，我们就把这个正整数称作质数。质数，又叫素数。
:::

### 素数条件

* 正整数:  质数都是正整数
* 大于1： 所有的质数都是大于1， 所以1 不是质数
* 有且只有1和自身两个约数； 任何一个质数都只有1和自身两个约数， 不能有1和自身以外的约数

### 示例

```

10以内的质数的寻找方法和过程。

（1）质数大于1，所以1不是质数。

（2）2的约数只有1、2。因为2的约数只有1和2自身，所以2是质数。

（3）3的约数只有1、3。因为3的约数只有1和3自身，所以3是质数。

（4）4的约数有1、2、4。因为4的约数除了1和自身外还有2，所以4不是质数。

（5）5的约数只有1、5。因为5的约数只有1和5自身，所以5是质数。

（6）6的约数有1、2、3、6。因为6的约数除了1和6自身外还有2、3，所以6不是质数。

（7）7的约数只有1、7。因为7的约数只有1和7自身，所以7是质数。

（8）8的约数有1、2、4、8。因为8的约数除了1和8自身外还有2、4，所以8不是质数。

（9）9的约数有1、3、9。因为9的约数除了1和9自身外还有3，所以9不是质数。

（10）10的约数有1、2、5、10。因为10的约数除了1和10自身外还有2、5，所以10不是质数。

综上可知，10以内的质数有4个，分别是：2、3、5、7。
```

输入： 10；  输出： 4
