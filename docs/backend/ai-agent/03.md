# 实战：智能客服机器人

本章基于 LangChain + RAG + Agent 构建一个可落地的企业智能客服系统。

## 需求分析

### 功能需求

1. **知识库问答**：基于企业文档回答产品/政策相关问题
2. **多轮对话**：记住上下文，支持追问
3. **人工转接**：复杂问题自动转人工
4. **意图识别**：区分咨询、投诉、售后等不同场景

### 技术选型

| 组件 | 选型 | 理由 |
|------|------|------|
| LLM | DeepSeek V3 | 性价比高、中文优秀 |
| 向量数据库 | PGVector | 与业务数据库共用基础设施 |
| 对话存储 | Redis | 高性能、过期策略 |
| 部署 | Docker + PM2 | 简单可靠 |

### 2. 知识库索引构建

```typescript
// src/knowledge/indexer.ts
import { OpenAIEmbeddings } from '@langchain/openai'
import { PGVectorStore } from '@langchain/community/vectorstores/pgvector'
import { RecursiveCharacterTextSplitter } from 'langchain/text_splitter'
import { DirectoryLoader } from 'langchain/document_loaders/fs/directory'
import { TextLoader } from 'langchain/document_loaders/fs/text'
import { config } from '../config'

export async function buildKnowledgeIndex() {
  const loader = new DirectoryLoader('./data', {
    '.txt': (path) => new TextLoader(path),
    '.md': (path) => new TextLoader(path),
  })

  const docs = await loader.load()

  const splitter = new RecursiveCharacterTextSplitter({
    chunkSize: 800,
    chunkOverlap: 100,
    separators: ['\n## ', '\n### ', '\n', '。', '！'],
  })

  const chunks = await splitter.splitDocuments(docs)

  const embeddings = new OpenAIEmbeddings({
    modelName: 'text-embedding-3-small',
    openAIApiKey: config.llm.apiKey,
    configuration: { baseURL: config.llm.baseURL },
  })

  await PGVectorStore.fromDocuments(chunks, embeddings, {
    postgresConnectionOptions: {
      connectionString: config.vectorStore.connectionString,
    },
    tableName: config.vectorStore.collectionName,
    columns: {
      idColumnName: 'id',
      vectorColumnName: 'vector',
      contentColumnName: 'content',
      metadataColumnName: 'metadata',
    },
  })

  console.log(`已索引 ${chunks.length} 个文档片段`)
}
```

### 3. 意图识别

```typescript
// src/chains/intent.ts
import { ChatOpenAI } from '@langchain/openai'
import { ChatPromptTemplate } from '@langchain/core/prompts'
import { StringOutputParser } from '@langchain/core/output_parsers'
import { config } from '../config'

export type Intent =
  | 'product_inquiry'    // 产品咨询
  | 'order_issue'        // 订单问题
  | 'complaint'          // 投诉
  | 'technical_support'  // 技术支持
  | 'human_transfer'     // 转人工
  | 'other'

const model = new ChatOpenAI({
  modelName: config.llm.modelName,
  temperature: 0,
  openAIApiKey: config.llm.apiKey,
  configuration: { baseURL: config.llm.baseURL },
})

const intentPrompt = ChatPromptTemplate.fromTemplate(`
分析用户消息，判断其意图。仅从以下标签中选择：
- product_inquiry: 询问产品信息、价格、功能
- order_issue: 订单查询、修改、退款
- complaint: 表达不满、投诉
- technical_support: 技术问题、使用故障
- human_transfer: 明确要求人工客服
- other: 其他

对话历史：
{history}

用户消息：{message}

只输出意图标签，不要解释：
`)

const intentChain = intentPrompt
  .pipe(model)
  .pipe(new StringOutputParser())

export async function detectIntent(
  message: string,
  history: string
): Promise<Intent> {
  const result = await intentChain.invoke({ message, history })
  return result.trim() as Intent
}
```

### 4. RAG 问答链

```typescript
// src/chains/rag.ts
import { ChatOpenAI, OpenAIEmbeddings } from '@langchain/openai'
import { PGVectorStore } from '@langchain/community/vectorstores/pgvector'
import { createRetrievalChain } from 'langchain/chains/retrieval'
import { createStuffDocumentsChain } from 'langchain/chains/combine_documents'
import { ChatPromptTemplate } from '@langchain/core/prompts'
import { config } from '../config'

const model = new ChatOpenAI({
  modelName: config.llm.modelName,
  temperature: 0.3,
  openAIApiKey: config.llm.apiKey,
  configuration: { baseURL: config.llm.baseURL },
})

const answerPrompt = ChatPromptTemplate.fromTemplate(`
你是某电商平台的智能客服。基于以下知识库内容回答用户问题。

规则：
1. 如果知识库中没有相关信息，回答"抱歉，我暂时无法回答这个问题，为您转接人工客服"
2. 回答简洁，控制在 200 字以内
3. 使用礼貌用语

知识库内容：
{context}

用户问题：{input}

请回答：
`)

export async function createRagChain() {
  const embeddings = new OpenAIEmbeddings({
    modelName: 'text-embedding-3-small',
    openAIApiKey: config.llm.apiKey,
    configuration: { baseURL: config.llm.baseURL },
  })

  const vectorStore = await PGVectorStore.initialize(embeddings, {
    postgresConnectionOptions: {
      connectionString: config.vectorStore.connectionString,
    },
    tableName: config.vectorStore.collectionName,
    columns: {
      idColumnName: 'id',
      vectorColumnName: 'vector',
      contentColumnName: 'content',
      metadataColumnName: 'metadata',
    },
  })

  const combineDocsChain = await createStuffDocumentsChain({
    llm: model,
    prompt: answerPrompt,
  })

  return createRetrievalChain({
    retriever: vectorStore.asRetriever({ k: 5 }),
    combineDocsChain,
  })
}
```

### 6. 主服务

```typescript
// src/index.ts
import express from 'express'
import { createRagChain } from './chains/rag'
import { detectIntent } from './chains/intent'
import { orderQueryTool, createTicketTool } from './tools'
import { RedisChatMemory } from './memory/redis'
import Redis from 'ioredis'
import { config } from './config'

const app = express()
app.use(express.json())

const redis = new Redis(config.redis)

// 初始化 RAG
const ragChainPromise = createRagChain()

app.post('/chat', async (req, res) => {
  const { sessionId, message } = req.body

  const memory = new RedisChatMemory(redis, sessionId)
  const history = await memory.loadMemoryVariables()
  const historyText = history.history
    .map((m: any) => `${m._getType()}: ${m.content}`)
    .join('\n')

  // 意图识别
  const intent = await detectIntent(message, historyText)

  let response: string

  switch (intent) {
    case 'order_issue': {
      // Agent 处理订单问题
      response = await handleOrderIssue(message, memory)
      break
    }
    case 'complaint':
    case 'human_transfer': {
      // 转人工
      await createTicketTool.invoke({
        userId: sessionId,
        issue: message,
        priority: intent === 'complaint' ? 'high' : 'medium',
      })
      response = '已为您转接人工客服，请稍候...'
      break
    }
    case 'product_inquiry':
    case 'technical_support':
    default: {
      // RAG 回答
      const ragChain = await ragChainPromise
      const result = await ragChain.invoke({ input: message })
      response = result.answer
    }
  }

  // 保存对话历史
  await memory.saveContext({ input: message }, { response })

  res.json({ intent, response })
})

app.listen(config.server.port, () => {
  console.log(`客服机器人启动在端口 ${config.server.port}`)
})
```
