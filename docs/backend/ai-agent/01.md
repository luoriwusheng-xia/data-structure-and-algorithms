# LangChain 快速入门

## 前置条件

- Node.js >= 18
- pnpm / npm
- DeepSeek API Key（或其他兼容 OpenAI 接口的模型服务）

## 环境准备

### 1. 获取 API Key

前往 [DeepSeek 开放平台](https://platform.deepseek.com/) 注册并创建 API Key。

### 2. 初始化项目

```bash
mkdir langchain-demo && cd langchain-demo
pnpm init
pnpm add @langchain/core @langchain/openai zod dotenv
pnpm add -D tsx typescript @types/node
```

### 3. 配置环境变量

```bash
# .env
DEEPSEEK_API_KEY=sk-xxxxxxxxxxxxxxxx
DEEPSEEK_BASE_URL=https://api.deepseek.com/v1
```

## 第一个程序：Hello LangChain

```typescript
// src/01-hello.ts
import { ChatOpenAI } from '@langchain/openai'
import { HumanMessage, SystemMessage } from '@langchain/core/messages'
import 'dotenv/config'

const model = new ChatOpenAI({
  modelName: 'deepseek-chat',
  temperature: 0.7,
  openAIApiKey: process.env.DEEPSEEK_API_KEY,
  configuration: {
    baseURL: process.env.DEEPSEEK_BASE_URL,
  },
})

async function main() {
  const messages = [
    new SystemMessage('你是一位专业的技术文档工程师。'),
    new HumanMessage('请用一句话解释什么是 LangChain。'),
  ]

  const response = await model.invoke(messages)
  console.log(response.content)
}

main()
```

运行：

```bash
npx tsx src/01-hello.ts
```

## 核心概念

### 1. Model（模型）

LangChain 通过统一的接口封装各种 LLM，支持 OpenAI、DeepSeek、Claude、本地模型等。

```typescript
// 流式输出
const stream = await model.stream([
  new HumanMessage('讲一个关于程序员的短故事'),
])

for await (const chunk of stream) {
  process.stdout.write(chunk.content)
}
```

### 2. Prompt（提示词）

使用 `ChatPromptTemplate` 构建结构化提示词：

```typescript
import { ChatPromptTemplate } from '@langchain/core/prompts'

const prompt = ChatPromptTemplate.fromMessages([
  ['system', '你是一位{role}，擅长用{style}风格回答。'],
  ['human', '{question}'],
])

const formatted = await prompt.formatMessages({
  role: '前端专家',
  style: '通俗易懂',
  question: '什么是虚拟 DOM？',
})

const response = await model.invoke(formatted)
```

### 3. Output Parser（输出解析器）

将模型输出解析为结构化数据：

```typescript
import { StructuredOutputParser } from '@langchain/core/output_parsers'
import { z } from 'zod'

const parser = StructuredOutputParser.fromZodSchema(
  z.object({
    answer: z.string().describe('问题的答案'),
    confidence: z.number().describe('置信度 0-1'),
    sources: z.array(z.string()).describe('参考来源'),
  })
)

const chain = prompt.pipe(model).pipe(parser)

const result = await chain.invoke({
  role: '前端专家',
  style: '通俗易懂',
  question: '什么是虚拟 DOM？',
})

console.log(result)
// { answer: '...', confidence: 0.95, sources: [...] }
```

### 4. Chain（链）

Chain 是将多个组件串联起来的管道：

```typescript
import { StringOutputParser } from '@langchain/core/output_parsers'

const translateChain = ChatPromptTemplate.fromTemplate(`
将以下文本翻译成{language}：

{text}
`).pipe(model).pipe(new StringOutputParser())

const result = await translateChain.invoke({
  language: '英语',
  text: '你好，世界',
})
```

## 完整示例：文章摘要生成器

```typescript
// src/02-summary.ts
import { ChatOpenAI } from '@langchain/openai'
import { ChatPromptTemplate } from '@langchain/core/prompts'
import { StringOutputParser } from '@langchain/core/output_parsers'
import 'dotenv/config'

const model = new ChatOpenAI({
  modelName: 'deepseek-chat',
  temperature: 0.3,
  openAIApiKey: process.env.DEEPSEEK_API_KEY,
  configuration: { baseURL: process.env.DEEPSEEK_BASE_URL },
})

const summaryPrompt = ChatPromptTemplate.fromTemplate(`
请对以下文章进行摘要，要求：
1. 摘要长度不超过 200 字
2. 保留核心观点和关键数据
3. 使用中文输出

文章：
{article}

摘要：
`)

const chain = summaryPrompt.pipe(model).pipe(new StringOutputParser())

async function summarize(article: string) {
  return chain.invoke({ article })
}

// 使用
const article = `
  近日，某科技公司发布了新一代 AI 芯片...
`
summarize(article).then(console.log)
```

## 小结

本章覆盖了 LangChain 最核心的四个概念：**Model**、**Prompt**、**Parser**、**Chain**。掌握这些基础后，可以继续学习 RAG、Memory、Agent 等高阶特性。
