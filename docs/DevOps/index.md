# DevOps 实战指南

本系列覆盖从代码到上线的完整 DevOps 链路，以 GitHub 为代码载体，结合 Claude Code、Vercel 等现代工具链，实现高效的前后端自动化部署。

## 前置条件

- 熟悉 Git / GitHub 基本操作
- 拥有 GitHub 账号
- 了解基础 Linux 命令

## 本章特色

- **代码驱动**：每个章节都包含可直接运行的配置和脚本
- **平台实战**：以 Vercel、GitHub Actions、Claude Code 为主线
- **端到端覆盖**：从 `git push` 到线上可访问的完整链路
- **Electron 打包**：覆盖 Windows / macOS / Linux 三平台自动构建
