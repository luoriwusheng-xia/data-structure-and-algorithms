### runtime-dom 安装依赖

在 项目根目录下执行如下命令， 模仿vue3 的 monorepo 仓库中安装 workspace中的依赖


```shell
pnpm add @vue/shared @vue/reactivity --workspace --filter @vue/runtime-dom
```
