# vue2.7.14 调试

```shell
git clone https://github.com/vuejs/vue.git
```

使用 pnpm 安装依赖

```shell
pnpm install
```

**环境**
```
node: 14.21.3 及以上
pnpm 7.2.0
```


```shell
npm run dev
```
打包文件，启用 `watch` 监听, 此时 dist 目录是会出现完整的 bundle

```
- dist
  - vue.common.js
  - vue.js
  - vue.runtime.common.js
  - vue.runtime.mjs
```


创建 demo目录， 直接使用相对目录，引入 `dist/vue.js` 即可
