## 一、WebGL 基础概念

### 1.1 什么是 WebGL

WebGL（Web Graphics Library）是一种 JavaScript API，用于在浏览器中渲染高性能的交互式 2D/3D 图形，无需插件。它基于 OpenGL ES 2.0/3.0，直接操作 GPU。

### 1.2 WebGL vs Canvas 2D

| 特性 | Canvas 2D | WebGL |
|------|-----------|-------|
| 渲染方式 | CPU 绘制 | GPU 加速 |
| 适用场景 | 2D 图形、简单动画 | 3D 场景、复杂着色器、大量顶点 |
| 性能 | 中等 | 极高（可处理百万级顶点） |
| 学习曲线 | 平缓 | 陡峭 |
| 控制力 | 有限 | 完全可控 |

### 1.3 获取 WebGL 上下文

```javascript
const canvas = document.getElementById('glcanvas');

// WebGL 1.0
const gl1 = canvas.getContext('webgl');

// WebGL 2.0（推荐）
const gl2 = canvas.getContext('webgl2');

// 兼容性降级
const gl = canvas.getContext('webgl2') || canvas.getContext('webgl');
if (!gl) {
  console.error('WebGL 不支持');
}
```

## 二、渲染管线

WebGL 渲染管线是将 3D 数据转换为 2D 屏幕像素的完整流程：

```
顶点数据 → 顶点着色器 → 图元装配 → 光栅化 → 片元着色器 → 帧缓冲 → 屏幕
```

### 2.1 顶点着色器（Vertex Shader）

- 输入：顶点位置、颜色、法线、纹理坐标等属性
- 输出：裁剪空间坐标（clip space），`gl_Position`
- 执行次数：每个顶点执行一次

### 2.2 片元着色器（Fragment Shader）

- 输入：光栅化后的插值数据
- 输出：像素颜色，`gl_FragColor`
- 执行次数：每个像素执行一次

### 2.3 最小可运行程序

```javascript
// 顶点着色器
const vsSource = `
  attribute vec4 aVertexPosition;
  void main() {
    gl_Position = aVertexPosition;
  }
`;

// 片元着色器
const fsSource = `
  precision mediump float;
  void main() {
    gl_FragColor = vec4(1.0, 0.0, 0.0, 1.0); // 红色
  }
`;

// 编译着色器
function loadShader(gl, type, source) {
  const shader = gl.createShader(type);
  gl.shaderSource(shader, source);
  gl.compileShader(shader);
  return shader;
}

const vertexShader = loadShader(gl, gl.VERTEX_SHADER, vsSource);
const fragmentShader = loadShader(gl, gl.FRAGMENT_SHADER, fsSource);

// 链接着色器程序
const program = gl.createProgram();
gl.attachShader(program, vertexShader);
gl.attachShader(program, fragmentShader);
gl.linkProgram(program);
gl.useProgram(program);
```

## 三、着色器语言 GLSL

### 3.1 数据类型

```glsl
// 标量
float f = 1.0;
int i = 1;
bool b = true;

// 向量
vec2 v2 = vec2(1.0, 2.0);
vec3 v3 = vec3(1.0, 2.0, 3.0);
vec4 v4 = vec4(1.0, 2.0, 3.0, 1.0); // RGBA

// 矩阵
mat2 m2 = mat2(1.0, 0.0, 0.0, 1.0);
mat4 m4 = mat4(1.0); // 单位矩阵
```

### 3.2 限定符

| 限定符 | 说明 | 示例 |
|--------|------|------|
| `attribute` | 顶点属性（仅顶点着色器） | `attribute vec3 aPosition;` |
| `uniform` | 全局常量（两着色器共用） | `uniform mat4 uModelMatrix;` |
| `varying` | 顶点着色器 → 片元着色器插值 | `varying vec2 vTexCoord;` |

> WebGL 2.0 中 `varying` 被 `in/out` 替代，对应 `attribute` 变为 `in`。

### 3.3 内置变量

- `gl_Position`：顶点着色器输出，裁剪空间坐标
- `gl_FragColor`：片元着色器输出，像素颜色
- `gl_FragCoord`：片元屏幕坐标
- `gl_PointSize`：点大小

## 四、缓冲区与顶点数据

### 4.1 创建顶点缓冲

```javascript
// 三角形顶点数据（x, y, z）
const positions = new Float32Array([
   0.0,  0.5, 0.0,  // 顶点1
  -0.5, -0.5, 0.0,  // 顶点2
   0.5, -0.5, 0.0,  // 顶点3
]);

// 创建缓冲区
const positionBuffer = gl.createBuffer();
gl.bindBuffer(gl.ARRAY_BUFFER, positionBuffer);
gl.bufferData(gl.ARRAY_BUFFER, positions, gl.STATIC_DRAW);

// 绑定到着色器属性
const positionLocation = gl.getAttribLocation(program, 'aVertexPosition');
gl.enableVertexAttribArray(positionLocation);
gl.vertexAttribPointer(positionLocation, 3, gl.FLOAT, false, 0, 0);
```

### 4.2 绘制调用

```javascript
// 清除画布
gl.clearColor(0.0, 0.0, 0.0, 1.0);
gl.clear(gl.COLOR_BUFFER_BIT);

// 绘制三角形
gl.drawArrays(gl.TRIANGLES, 0, 3);

// 其他图元类型
// gl.POINTS         // 点
// gl.LINES          // 线
// gl.LINE_STRIP     // 线带
// gl.LINE_LOOP      // 线圈
// gl.TRIANGLE_STRIP // 三角带
// gl.TRIANGLE_FAN   // 三角扇
```

## 五、纹理映射

### 5.1 加载与绑定纹理

```javascript
// 创建纹理
const texture = gl.createTexture();
gl.bindTexture(gl.TEXTURE_2D, texture);

// 设置参数
gl.texParameteri(gl.TEXTURE_2D, gl.TEXTURE_WRAP_S, gl.CLAMP_TO_EDGE);
gl.texParameteri(gl.TEXTURE_2D, gl.TEXTURE_WRAP_T, gl.CLAMP_TO_EDGE);
gl.texParameteri(gl.TEXTURE_2D, gl.TEXTURE_MIN_FILTER, gl.LINEAR);
gl.texParameteri(gl.TEXTURE_2D, gl.TEXTURE_MAG_FILTER, gl.LINEAR);

// 加载图像
const image = new Image();
image.onload = () => {
  gl.bindTexture(gl.TEXTURE_2D, texture);
  gl.texImage2D(gl.TEXTURE_2D, 0, gl.RGBA, gl.RGBA, gl.UNSIGNED_BYTE, image);
};
image.src = 'texture.png';
```

### 5.2 纹理坐标

```glsl
// 顶点着色器
attribute vec2 aTexCoord;
varying vec2 vTexCoord;

void main() {
  gl_Position = /* ... */;
  vTexCoord = aTexCoord;
}

// 片元着色器
precision mediump float;
varying vec2 vTexCoord;
uniform sampler2D uSampler;

void main() {
  gl_FragColor = texture2D(uSampler, vTexCoord);
}
```

## 六、矩阵变换与坐标系

### 6.1 坐标变换流水线

```
模型空间 → 模型矩阵(Model) → 世界空间
世界空间 → 视图矩阵(View) → 观察空间
观察空间 → 投影矩阵(Projection) → 裁剪空间
裁剪空间 → 透视除法 → NDC
NDC → 视口变换 → 屏幕空间
```

### 6.2 MVP 矩阵

```glsl
// 顶点着色器
uniform mat4 uModelMatrix;
uniform mat4 uViewMatrix;
uniform mat4 uProjectionMatrix;
attribute vec4 aVertexPosition;

void main() {
  gl_Position = uProjectionMatrix * uViewMatrix * uModelMatrix * aVertexPosition;
}
```

### 6.3 常用矩阵库

推荐使用 `gl-matrix`：

```javascript
import { mat4, vec3 } from 'gl-matrix';

// 创建投影矩阵
const projection = mat4.create();
mat4.perspective(projection, Math.PI / 4, canvas.width / canvas.height, 0.1, 100.0);

// 创建视图矩阵
const view = mat4.create();
mat4.lookAt(view, [0, 0, 5], [0, 0, 0], [0, 1, 0]);

// 创建模型矩阵
const model = mat4.create();
mat4.rotateY(model, model, Math.PI / 4);

// 传递到着色器
gl.uniformMatrix4fv(uModelLocation, false, model);
```

## 七、帧缓冲与离屏渲染

### 7.1 创建帧缓冲对象（FBO）

```javascript
// 创建纹理附件
const texture = gl.createTexture();
gl.bindTexture(gl.TEXTURE_2D, texture);
gl.texImage2D(gl.TEXTURE_2D, 0, gl.RGBA, width, height, 0, gl.RGBA, gl.UNSIGNED_BYTE, null);

// 创建帧缓冲
const framebuffer = gl.createFramebuffer();
gl.bindFramebuffer(gl.FRAMEBUFFER, framebuffer);
gl.framebufferTexture2D(gl.FRAMEBUFFER, gl.COLOR_ATTACHMENT0, gl.TEXTURE_2D, texture, 0);

// 检查完整性
if (gl.checkFramebufferStatus(gl.FRAMEBUFFER) !== gl.FRAMEBUFFER_COMPLETE) {
  console.error('帧缓冲不完整');
}

// 渲染到纹理
gl.bindFramebuffer(gl.FRAMEBUFFER, framebuffer);
// ... 绘制 ...

// 切换回屏幕
gl.bindFramebuffer(gl.FRAMEBUFFER, null);
```

### 7.2 FBO 应用场景

- **后处理特效**：模糊、泛光、色调映射
- **阴影映射**：生成深度纹理
- **反射/折射**：离屏渲染环境贴图
- **拾取（Picking）**：渲染物体 ID 到纹理

## 小结

| 概念 | 要点 |
|------|------|
| 渲染管线 | 顶点 → 图元 → 光栅化 → 片元 → 像素 |
| 着色器 | 顶点着色器处理几何，片元着色器处理颜色 |
| GLSL | 类 C 语法，向量/矩阵操作原生支持 |
| 缓冲区 | `ARRAY_BUFFER` 存顶点数据，`ELEMENT_ARRAY_BUFFER` 存索引 |
| 纹理 | `sampler2D` 采样，`texImage2D` 上传 |
| MVP 矩阵 | `P * V * M` 将模型顶点变换到裁剪空间 |
| FBO | 离屏渲染的核心，后处理和阴影的基石 |
