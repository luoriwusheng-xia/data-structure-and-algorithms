### 位运算

> 位运算是一种在计算机中直接对二进制位进行操作的运算方式

1. 快速：位运算通常比算术运算和逻辑运算更快，因为它们直接操作二进制位，不涉及复杂的数学运算或逻辑判断。
2. 高效：位运算可以用于执行一些特定的位操作，如位移、位与、位或、位异或等，这可以使某些复杂计算问题的实现更加高效。
3. 节省空间：进行位运算时，位数据可以被有效地压缩在一个或多个机器字中，这样可以节省空间和存储成本。
4. 适用于底层操作：位运算常用于底层编程，如位掩码、位标志、位图等。它们在处理硬件设备、嵌入式系统、图形处理等方面发挥着重要作用。
5. 不易理解：位运算通常较为底层和抽象，使用它们的代码可能较难理解和维护。对于一般的业务逻辑，合适的算术和逻辑运算更易于理解。


位运算的特点是， 2个相同的数字进行位运算， 一定是 0;  0 和其他非0的数字进行位运算，一定是数字本身

异或`^` 满足`交换律`

`a ^ b = b ^ a`,  也就是说， 一堆数字异或， 可以推演成如下

`1 ^ 2 ^ 1 ^ 3 ^ 2 = 1 ^ 1 ^ 2 ^ 2 ^ 3 = 3`


### 位运算举例
#### 位与（AND）运算：

> 位与运算符将两个操作数的对应位进行逻辑与操作。只有在对应位都为1时，结果位才为1。

```javascript
a = 5  // 二进制  0101 = 2^0 + 0*2^0 + 1*2^2 = 5
b = 3 // 二进制   0011

result = a & b  // 0001  , 十进制的 1
0101
0011
----
0001
```


#### 位或（OR）运算：
> 位或运算符将两个操作数的对应位进行逻辑或操作。只要对应位中有一个为1，结果位就为1。

```javascript
a = 5  # 二进制表示为 0101
b = 3  # 二进制表示为 0011
result = a | b  # 二进制结果为 0111，即十进制的 7

0101
0011
----
0111

```

#### 位异或（XOR）运算
> 位异或运算符将两个操作数的对应位进行逻辑异或操作。当对应位只有一个为1时，结果位为1。

```javascript
a = 5  # 二进制表示为 0101
b = 3  # 二进制表示为 0011
result = a ^ b  # 二进制结果为 0110，即十进制的 6

0101
0011
-----
0110

```


#### 位取反（NOT）运算
> 位取反运算符将操作数的每个位进行取反操作，即0变为1，1变为0。

```javascript
a = 5  # 二进制表示为 0101
result = ~a  # 二进制结果为 1010，即十进制的 -6（取反后加1）

0101
-----
1010

```
