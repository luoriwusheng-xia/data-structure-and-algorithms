# 位运算

> 左移和右移都是操作的二进制数据

### 概念

先熟悉基础的二进制

```
0000 0000 --》 十进制就是  0
从右到左，权重依次为

2^0  2^1  2^2  2^3  2^4  2^5 2^6  2^7


则计算结果 = 0 * 2^0 + 0* 2^1 + 0*2^2 +0* 2^3 +0* 2^4 +0* 2^5 +0*2^6 +0* 2^7 = 0

```

```
2^0 = 1
2^1 = 2
2^2 = 4
2^3 = 8
2^4 = 16
2^5 = 32
2^6 = 64
2^7 = 128
2^8 = 256
....
```

js里面表示 **次方** 使用 `Math.pow(2,6)`

**案例1**

现在我们讨论 十进制的 8， 使用二进制怎么表示

```
0000 1000
```

**案例2**

十进制的5，使用二进制怎么表示

```
0000 0101
```

```
    除数  余数
5/2 = 3....1
2/2 = 1... 0
1/2 = 0... 1

```
从下往上，就是101 就是二进制

**案例3**

十进制23，使用二进制怎么表示

```
23/2 = 11 ... 1
11/2 = 5 .... 1
5/2 = 2 .... 1
2/2 = 1 .... 0
1/2 = 0 .... 1
```

所以，二进制为  0001 0111

### js 二进制和十进制互转

```javascript
// 十进制 27转成二进制  '11011'
Number(27).toString(2)

Number(23).toString(2) // '10111'
```

```javascript
// 二进制转十进制

parseInt("11011", 2) // 27
parseInt("10111", 2) // 23
```


### 左移

\<<< 表示左移， 左移，是数值 **变大**



### 右移
\>>> 表示右移， 右移，是数值 **变小**
